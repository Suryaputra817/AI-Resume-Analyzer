## ✨ Introduction

AI Resume Analyzer is a comprehensive frontend web application built to help job seekers improve their resumes. It allows users to upload their resumes in PDF format, analyzes the content against Applicant Tracking System (ATS) standards using AI, and provides visually engaging, custom feedback. 

With built-in authentication and cloud storage powered by Puter.js, users can safely manage and evaluate their resumes against tailored job listings.

## ⚙️ Tech Stack

- **[React](https://react.dev/)**: Core library for building the reusable user interface.
- **[React Router](https://reactrouter.com/)**: Handles seamless navigation and routing across the application.
- **[Puter.js](https://puter.com/)**: Client‑side SDK that provides serverless authentication, cloud storage, and AI processing straight from the browser.
- **[React Dropzone](https://react-dropzone.js.org/)**: Provides intuitive, drag-and-drop file uploading functionality.
- **[TypeScript](https://www.typescriptlang.org/)**: Ensures robust, error-free code through static typing.
- **[Vite](https://vite.dev/)**: Next-generation frontend tooling for lightning-fast server starts and builds.

## 🔋 Key Features

👉 **Secure Authentication**: Handle login and registration entirely in the browser using Puter.js.  
👉 **Drag-and-Drop Uploads**: Easily upload PDF resumes with a smooth drag-and-drop interface.  
👉 **AI Resume Matching**: Evaluate resumes against specific job listings to generate an accurate ATS score and personalized feedback.  
👉 **Visual Analytics**: Interactive score circles, gauges, and detailed accordions make feedback easy to read.  
👉 **Cloud Storage**: Safely store and manage your uploaded resumes and analysis data.
